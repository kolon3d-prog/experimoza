# experimoza 

basically, this is a quite spontaneous project. i've found out about hack club yesterday and i was **SOOOO** motivated to code something. i've looked around everywhere in my room until i've looked on my moza r3 wheel.

the whole idea is to create simple chain between ai and force feedback. you only need to send one image to ai.py file. then you run moza_effect_mvp.exe and force feedback works on your steering wheel.

## surface photo -> AI(gemini api) -> C++ script -> Moza R3 -> PROFIT!!
